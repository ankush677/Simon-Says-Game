# 🎮 Simon Says Game

A fun and interactive Simon Says memory game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- 🎨 Modern UI with Glassmorphism
- 🟢 Dynamic color sequence generation
- 📈 Score tracking
- 🏆 Best score tracking
- ✨ Smooth button animations
- 💥 Game-over shake effect
- ⌨️ Keyboard controls

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 🎯 How to Play

1. Press any key to start.
2. Watch the highlighted color sequence.
3. Repeat the sequence by clicking the buttons.
4. The sequence grows longer after every correct round.
5. Game ends when you click the wrong button.
